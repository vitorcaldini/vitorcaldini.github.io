Parâmetros de entrada¶

O modelo recebe como entrada 3 parmetros:

  • A matriz de distncia entre as localizaes;
  • A quantidade de veculos;
  • O ndice da matriz que corresponde ao depsito.

A matriz de distncia pode ser inserida diretamente ou via API da Google a partir dos endereos.

  • Para saber como conectar pela API da Google tem este guia: Google Distance Matrix API.
  • Para saber como criar uma matriz de distncia a partir de uma tabela de lagitudes e longitudes, recomendo este artigo.

Vale lembrar que a matriz de distncia da API do google retorna distncia real entre os endereos. Os clculos de distncia a partir de arcos, pares de valores (x,y) entre outros retornam as ditncias de acordo com o mtodo aplicado: euclidiana, manhattan, etc.

O cdigo a seguir cria o gerenciador de ndice (manager) e o modelo de rota (routing). O mtodo manager.IndexToNode converte os ndices internos do solver (que voc pode ignorar com segurana) em nmeros de localizaes. Os nmeros de localizao correspondem aos ndices da matriz de distncia.

As entradas para RoutingIndexManager so:

  • O nmero de linhas da matriz de distncia, que o nmero de locais (incluindo o depsito).
  • O nmero de veculos no problema.
  • O n correspondente ao depsito.

Para usar o solver de rotas, voc precisa criar um callback de distncia: uma funo que pega qualquer par de locais e retorna a distncia entre eles. Por isso a entrada em formato de matriz, facilita muito o callback.

A funo distance_callback faz exatamente isso: converte o ndice da matriz no ns (valores de distncia).

A funo RegisterTransitCallback registra o callback no solver como transit_callback_index. O callback aceita dois ndices, from_index e to_index, e retorna a entrada correspondente da matriz de distncia.

O avaliador do custo do arco diz ao solucionador como calcular o custo da viagem entre quaisquer dois locais - em outras palavras, o custo da linha (ou rota) que os une no grfico para o problema. O cdigo a seguir define o avaliador de custo do arco.

Neste exemplo, o avaliador do custo do arco transit_callback_index, que a referncia interna do solucionador para o o callback de distncia. Isso significa que o custo da viagem entre quaisquer dois locais apenas a distncia entre eles. Em outros casos, este custo pode ser alterado de acordo outros fatores envolvidos.

Voc tambm pode definir vrios avaliadores de custo de arco que dependem de qual veculo est viajando entre os locais, usando o mtodo routing.SetArcCostEvaluatorOfVehicle(). Por exemplo, se os veculos tm velocidades diferentes, voc pode definir o custo da viagem entre os locais como a distncia dividida pela velocidade do veculo - em outras palavras, o tempo de viagem.

O cdigo define a primeira estratgia de soluo para PATH_CHEAPEST_ARC, que cria uma rota inicial para o solver, adicionando repetidamente arestas com o menor peso que no conduzem a um n visitado anteriormente (diferente do depsito). Isso faz com que o programa no precise testas todas as combinaes possveis.

def print_solution(manager, routing, solution):
    """Prints solution on console."""
    print('Menor distncia percorrida: {} m'.format(solution.ObjectiveValue()))
    index = routing.Start(0)
    plan_output = 'Rota do veculo 0:\n'
    route_distance = 0
    while not routing.IsEnd(index):
        plan_output += ' {} ->'.format(manager.IndexToNode(index))
        previous_index = index
        index = solution.Value(routing.NextVar(index))
        route_distance += routing.GetArcCostForVehicle(previous_index, index, 0)
    plan_output += ' {}\n'.format(manager.IndexToNode(index))
    print(plan_output)
    plan_output += 'Route distance: {}miles\n'.format(route_distance)

def get_routes(solution, routing, manager):
  """Get vehicle routes from a solution and store them in an array."""
  # Get vehicle routes and store them in a two dimensional array whose
  # i,j entry is the jth location visited by vehicle i along its route.
  routes = []
  for route_nbr in range(routing.vehicles()):
    index = routing.Start(route_nbr)
    route = [manager.IndexToNode(index)]
    while not routing.IsEnd(index):
      index = solution.Value(routing.NextVar(index))
      route.append(manager.IndexToNode(index))
    routes.append(route)
  return routes

def print_solution(data, manager, routing, solution):
    """Imprime a soluo no console"""
    max_route_distance = 0
    for vehicle_id in range(data['num_vehicles']):
        index = routing.Start(vehicle_id)
        plan_output = 'Rota do veculo {}:\n'.format(vehicle_id)
        route_distance = 0
        while not routing.IsEnd(index):
            plan_output += ' {} -> '.format(manager.IndexToNode(index))
            previous_index = index
            index = solution.Value(routing.NextVar(index))
            route_distance += routing.GetArcCostForVehicle(
                previous_index, index, vehicle_id)
        plan_output += '{}\n'.format(manager.IndexToNode(index))
        plan_output += 'Distancia da rota: {}m\n'.format(route_distance)
        print(plan_output)
        max_route_distance = max(route_distance, max_route_distance)
    print('Distncia mxima entre as rotas: {}m'.format(max_route_distance))


def main():
    """Solver"""
    # Cria uma varivel com os dados
    data = create_data_model()

    # Manager de rota
    manager = pywrapcp.RoutingIndexManager(len(data['distance_matrix']),
                                           data['num_vehicles'], data['depot'])

    # Modelo de rota
    routing = pywrapcp.RoutingModel(manager)


    # Callback de distncia
    def distance_callback(from_index, to_index):
        """Returns the distance between the two nodes."""
        # Para pescar as distncias na matriz
        from_node = manager.IndexToNode(from_index)
        to_node = manager.IndexToNode(to_index)
        return data['distance_matrix'][from_node][to_node]

    transit_callback_index = routing.RegisterTransitCallback(distance_callback)

    # Definio do custo - nesse caso custo = distncia
    routing.SetArcCostEvaluatorOfAllVehicles(transit_callback_index)

    # Restrio de distncia
    dimension_name = 'Distance'
    routing.AddDimension(
        transit_callback_index,
        0, 
        50000,  # maximo
        True,  # start cumul to zero
        dimension_name)
    distance_dimension = routing.GetDimensionOrDie(dimension_name)
    distance_dimension.SetGlobalSpanCostCoefficient(100)

    # Primeira solues heuristicas
    search_parameters = pywrapcp.DefaultRoutingSearchParameters()
    search_parameters.first_solution_strategy = (
        routing_enums_pb2.FirstSolutionStrategy.PATH_CHEAPEST_ARC)

    # Solver
    solution = routing.SolveWithParameters(search_parameters)

    # Imprimir no console
    if solution:
        print_solution(data, manager, routing, solution)

if __name__ == '__main__':
    main()
